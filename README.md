# Variables and Control Structures

A Python project for learning and practicing fundamental programming concepts including variables, data types, and control structures.

## Overview

This repository contains educational Python scripts demonstrating core programming concepts such as:
- **Variables**: Storing and manipulating data in programs
- **Data Types**: Working with strings and other primitive types
- **Control Structures**: Logic flow and decision-making in code

## Project Structure

- `Variables-storing-Data in programs.py` - Introduction to variables and string data storage

## Getting Started

### Prerequisites

- Python 3.x installed on your system

### Running the Scripts

To run any Python script in this project:

```bash
python "Variables-storing-Data in programs.py"
```

## Learning Objectives

This project helps you understand:
- How to declare and initialize variables
- Working with string data types
- Printing and displaying variable values
- Basic Python syntax and conventions

## Example

```python
name = "John Doe"
fact = "A traffic jam lasted lasted for more than 10 days, with cars only moving 0.6 miles a day."
address = "77 Winchester Lane"
empty_str = ""

print(name)
print(fact)
print(address)
print(empty_str)
```
